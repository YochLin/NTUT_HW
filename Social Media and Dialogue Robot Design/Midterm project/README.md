# 期中專案
## 主題介紹
* 本專案為模擬診所的 Line 聊天機器人，能即時提供用戶做線上掛號，減輕病人在門診等待的時刻。
* 提供醫療紀錄查詢、營業時間查詢、門診位置查詢…等。

## 系統 / 軟體
<ol>
<li> firebase
<li> dialogflow
<li> typescript
<li> nodeJs
</ol>

## 系統架構

## 角色說明
* 一般用戶：聊天掛號、看診紀錄查詢、診所記錄查詢
* 管理者：查看用戶看診紀錄、統計各科別掛號次數
